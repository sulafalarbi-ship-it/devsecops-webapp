# DevSecOps Web Application

This repository contains two versions of the web application:

## V1 – Basic Version (No Security)
- File: v1/app_v1.py
- Simple login and welcome page
- No password validation or security checks

## V2 – Secure Version
- File: v2/app_v2.py
- Authentication with password validation
- Improved security controls

## How to Run
```bash
python v1/app_v1.py
python v2/app_v2.py
